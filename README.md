MAFIA: Identity Protocol is a cinematic role-assignment web app for the Mafia party game.

Built with:

- Vanilla JavaScript
- TailwindCSS
- Crypto-secure role shuffling (Web Crypto API)

Features:

- Government dossier-style UI
- Terminal typing animation
- Suspenseful Mafia reveal sequence
- Dynamic role badges (Approved / Record Rejected)
- Replayable role viewing system

Designed for offline party gameplay.
